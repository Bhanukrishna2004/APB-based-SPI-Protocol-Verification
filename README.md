# APB-based-SPI-Protocol-Verification
APB-Based SPI Controller Verification using UVM. Developed a SystemVerilog UVM testbench with sequence, driver, monitor, agent, and scoreboard to verify SPI functionality over the APB interface. Validated read/write operations and SPI communication (MOSI, MISO, SCLK, SS) using constrained random testing and waveform analysis.
